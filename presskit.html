Overview

Snippetty is a code snippet manager designed to assist speakers during live code demos. It lives in the macOS system menu and allows the audience to focus on the sequence of steps and ideas behind the code instead of waiting for code to be type in.

Early Snippetty prototypes were tested at 40+ tech events by creator Marin Todorov to find the best workflow for live code demos powered by a snippet manager. Snippetty is a available as a free limited demo version to try before buying.
